# iordek.research20206

Дослідження

Веб-додаток для збору анкетних даних учасників дослідження зі збереженням в Airtable.

## Документи проекту

- `data-structure.md` — структура полів анкети (джерело правди)
- `index.html` — веб-форма (режим заповнення та режим друку)
- `image-to-airtable.md` — інструкція для AI з обробки фото анкет
- `CLAUDE.md` — архітектура та процес розробки
- `CHANGELOG.md` — журнал змін

# Журнал змін

> **Цей файл містить**: хронологію змін у проекті по версіях.
> **Цей файл не містить**: поточного опису структури даних, форми чи процесу розробки — для цього дивись `data-structure.md`, `index.html` та `CLAUDE.md` відповідно.

## [0.48.0] — 17 червня 2026 року

### Додано
- **`dynamics.html`**: нова сторінка "Динаміка" — три панелі поруч для кожного учасника: Первинне (з таблиці Учасники), Контрольне та Вихідне (з таблиці Опитування)
- **`index.html`**: посилання на сторінку "Динаміка" у розділі "Перегляд даних"

### Виправлено
- **`dynamics.html`**: пошук використовує `returnFieldsByFieldId=true` — поля повертаються за ID, а не за назвою
- **`dynamics.html`**: первинне опитування читається з таблиці Учасники (patient.html), контрольне та вихідне — з таблиці Опитування

## [0.47.0] — 16 червня 2026 року

### Додано
- **Усі 5 форм**: індикатор дати народження поруч з полем пацієнта — показує ДН при введенні прізвища або Study ID (після виходу з поля). Зелений — знайдено, помаранчевий — кілька збігів, червоний — не знайдено
- **Усі 5 форм**: не потребує пароля для пошуку пацієнта; використовується окремий ключ тільки для читання

### Виправлено
- **`assessment.html`**: прибрано відладковий код; форма у фінальному стані

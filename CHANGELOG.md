# Журнал змін

## [0.1.0] — 11 червня 2026 року

### Додано
- ✨ **Пілотна версія проекту** з мінімальною структурою даних
- 📄 **data-structure.md** — описання двох базових полів:
  - Кодовий нікнейм пацієнта (текст)
  - Стать пацієнта (вибір: М/Ж/Інша/Не вказано)
- 🌐 **index.html** — веб-форма для збору даних:
  - Режим заповнення (звичайна форма)
  - Режим друку (для друкування та заповнення від руки)
  - Локальне збереження даних (localStorage)
- 🤖 **image-to-airtable.md** — інструкція для AI:
  - Розпізнавання даних з фотографій
  - Контроль якості з уточнюючими запитаннями
  - Запис в Airtable через MCP
- 📋 **CLAUDE.md** — документація проекту та інструкції для розробки
- 📝 **CHANGELOG.md** — цей файл

### Технічні деталі
- Форма використовує HTML5 та ванільний JavaScript
- Стилізація адаптована для вебу та друку
- Дані зберігаються локально в localStorage
- Готово до інтеграції з Airtable MCP
